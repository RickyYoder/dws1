Why Chrome Apps Could Be The Next Frontier

by Site Admin (10/29/2015)

Let's face it. We all love Google Chrome. No? Okay okay fine. At least 25% of us love Google Chrome. It's a great browser with a lot of features such as browser extensions, intricate settings, and the ability to sync our history, tabs, and other browsing data on all of our devices. Not to mention Chrome's availability on nearly all platforms: Windows, Mac, Linux, Android, and iOS. Between fast browsing, great features, and it's ubuquity, Chrome has become a staple for web browsers worldwide. And in the development world, it's no different.

When testing the latest and greatest browsing features, users are always encouraged to use Chrome. That's because it scores the highest on HTML5Test.com.

But Google wants more than a simple Internet browser. They want an entire platform of apps that run on Chrome. That's where Chrome Apps come in.

But What ARE Chrome Apps?

Chrome Apps are just like any other web app, with a few big exceptions. To sum them up...

  1. Chrome Apps can access device hardware such as serial ports, Bluetooth, and USB ports
  2. Unlike common misconceptions, some Chrome Apps CAN run offline
  3. Chrome Apps run independently from a web browser. They run in their own windows, just like a native application
  4. Chrome Apps are much safer than native apps, considering Google has put great efforts into making the API safe
